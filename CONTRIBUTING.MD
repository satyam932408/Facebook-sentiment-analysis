# Contributing to Facebook Sentiment Analysis

Thanks for helping improve this project! Small documentation or example fixes are especially welcome.

How to contribute (quick):
1. Fork the repository.
2. Create a branch for your change:
   `git checkout -b feat/your-change`
3. Make small, focused changes (docs, examples, minor fixes).
4. Commit with a clear message:
   `git commit -m "docs: improve README or add example"`
5. Push to your fork:
   `git push origin feat/your-change`
6. Open a Pull Request against `satyam932408/Facebook-sentiment-analysis` main branch.

Testing locally (optional):
- Create a Python venv and install requirements (if any).
- Run the analysis script or notebook as described in the README.

If you want help reviewing PRs, ping the repository owner.
